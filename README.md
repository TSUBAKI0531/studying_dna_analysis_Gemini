## 2. Gene-Expression-Analysis: 遺伝子発現データの解析と機械学習の実践

このパートでは、次世代シーケンサー（NGS）の解析結果などで得られる「遺伝子発現量」を模したデータセットを用い、バイオインフォマティクスにおける標準的なデータパイプラインを実践しました。
なおこのパート作成にはGeminiを利用しています。

### 解析内容
1. **データ取得**: Scikit-learn提供の「乳がん（Breast Cancer）」データセットを使用（30種類の特徴量を遺伝子発現量に見立てて解析）。
2. **データ前処理**: 
    - 遺伝子データの特性に合わせ、`log1p` による対数変換のシミュレーションを実施。
    - `StandardScaler` を用いて各特徴量を標準化（平均0, 分散1）し、機械学習に適した形に加工。
3. **可視化（次元圧縮）**:
    - **主成分分析（PCA）**を実行し、高次元データを2次元に圧縮。
    - 散布図により、疾患群（悪性）と正常群（良性）の明確なクラスタリングを視覚的に確認。
4. **機械学習**:
    - ランダムフォレストを用いて分類モデルを構築し、高精度な予測を実現。
    - **Feature Importance（特徴量の重要度）**を算出し、判定に大きく寄与した「マーカー遺伝子候補」を特定。
5. **仮想データ検証**:
    - 悪性・良性の平均的な特徴量を持つ仮想サンプルを作成し、モデルの判定の妥当性と確信度（Probability）を検証。

### 学習のポイント
- 多次元データの「見え方」をPCAで把握する重要性。
- モデルの精度だけでなく、「なぜその結果になったか」という生物学的解釈（重要度）の紐付け。
